오픈 소스 브라우저 목록

Chromium

Google Chrome의 기반이 되는 오픈 소스 브라우저.

운영체제: Windows, Linux, Mac

설치 용량: 약 200MB

다운로드

Git 설치: git clone https://chromium.googlesource.com/chromium/src.git

Mozilla Firefox

강력한 보안과 커스터마이징 기능을 제공하는 오픈 소스 브라우저.

운영체제: Windows, Linux, Mac

설치 용량: 약 250MB

다운로드

Git 설치: git clone https://github.com/mozilla/gecko-dev.git

Pale Moon

운영체제: Windows, Linux

설치 용량: 약 70MB

다운로드

Git 설치: git clone https://github.com/MoonchildProductions/Pale-Moon.git

다운로드

Git 설치: git clone https://github.com/MoonchildProductions/UXP.git
