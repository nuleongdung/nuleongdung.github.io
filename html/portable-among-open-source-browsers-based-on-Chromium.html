크로미움 기반 오픈소스 포터블 브라우저 목록

 

Chromium

장점

  • 오픈 소스이므로 누구나 검토하고 수정 가능
  • 빠른 속도와 높은 안정성
  • 구글 생태계와의 뛰어난 호환성

단점

  • 개인정보 보호 설정이 부족할 수 있음
  • 자원 소모가 많음
  • 구글 서비스 의존도가 높음

주요 소스코드 문제점

  • 거대한 코드베이스로 인한 관리 어려움
  • 오래된 코드의 잔존 가능성
  • 취약점 패치 속도에 따라 보안 위험

보안 등급

8/10

Brave

장점

  • 광고 차단 기능 기본 내장
  • 빠른 페이지 로딩 속도
  • 높은 프라이버시 보호 기능

단점

  • 일부 사이트와의 호환성 문제
  • 광고 기반 수익 모델에 대한 논란
  • 사용자 기반이 적어 피드백 수집 어려움

주요 소스코드 문제점

  • 비교적 새로운 코드로 검증 시간 부족
  • 광고 차단 기능이 일부 사이트에서 오작동
  • 빠른 업데이트 주기로 인한 잠재적 버그

보안 등급

9/10

Vivaldi

장점

  • 고도로 사용자화 가능한 UI
  • 생산성 기능이 풍부 (노트, 탭 스택)
  • 강력한 개인정보 보호 기능

단점

  • 복잡한 인터페이스로 인한 초기 학습 곡선
  • 다소 느린 성능
  • 모바일 버전의 기능 부족

주요 소스코드 문제점

  • 크로미움 기반으로 독창적 코드 제한
  • 커스터마이징 코드의 복잡성
  • 새로운 기능 추가 시 발생하는 잠재적 버그

보안 등급

  • 무료 VPN 기능 내장

  • 빠른 성능과 배터리 절약 모드

단점

  • 중국 소유권에 대한 보안 우려
  • 광고 차단 기능이 제한적
  • 사용자 커뮤니티가 작음

주요 소스코드 문제점

  • 폐쇄된 코드로 인한 투명성 부족
  • VPN의 신뢰성 문제
  • 크로미움 업데이트 지연 가능성
