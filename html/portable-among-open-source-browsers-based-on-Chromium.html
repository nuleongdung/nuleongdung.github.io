크로미움 기반 오픈소스 포터블 브라우저 목록

크로미움 기반 브라우저

Chromium

장점

오픈소스 기반으로 누구나 코드 수정 가능, 빠른 속도 및 최신 웹 표준 적용.

단점

자동 업데이트 기능 없음, 일부 기능(코덱, DRM 등) 제한됨.

이슈

구글 서비스 종속성 문제, 일부 기능 크롬과 차이 존재.

Brave

장점

광고 및 트래커 차단 기본 제공, 빠른 속도.

단점

일부 웹사이트에서 문제 발생 가능.

이슈

기본 검색 엔진 및 광고 수익 모델 논란.

Vivaldi

장점

높은 커스터마이징 가능, 다양한 내장 기능.

단점

다소 무거운 성능.

이슈

기능이 많아 학습 곡선이 필요함.

Opera

장점

VPN 및 배터리 절약 모드 제공.

단점

중국 기업 인수 후 개인정보 보호 논란.

이슈

일부 기능이 불필요하게 많음.

Epic Privacy Browser

장점

추적 방지 기능 강화, 프라이버시 보호.

단점

일부 기능 제한적.

이슈

오픈소스 아님, 신뢰성 논란.

SRWare Iron

장점

구글 추적 기능 제거, 빠른 속도.

단점

업데이트가 느림.

이슈

보안 패치 적용이 느릴 수 있음.

장점

보안 기능 강화.

단점

추가 기능 부족.

이슈

장점

구글 서비스 완전 제거.

단점

자동 업데이트 없음.

이슈

일반 사용자에게 설정이 어려울 수 있음.

장점

빠른 속도, 러시아 시장 최적화.

단점

개인정보 보호 논란.

이슈

러시아 정부와의 연계 가능성.

장점

메모리 관리 최적화.

단점

이슈

이슈
