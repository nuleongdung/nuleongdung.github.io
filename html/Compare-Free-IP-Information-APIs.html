다음은 JavaScript를 사용하여 ipinfo.io API를 호출하고 IP 주소 정보를 가져오는 예제입니다.

                
// 현재 사용자의 IP 주소 정보를 가져오는 예제

.then(response => {
    if (!response.ok) {
    throw new Error('네트워크 응답이 좋지 않습니다.');
    }
    return response.json();
})
.then(data => {
    console.log('IP 주소 정보:', data);
    // 필요한 정보를 화면에 표시하거나 다른 작업 수행
    document.getElementById('sf-ipinfo-result').innerHTML = `
    <h4>결과 표시 영역</h4>
    <p>IP: ${data.ip}</p>
    <p>도시: ${data.city}</p>
    <p>지역: ${data.region}</p>
    <p>국가: ${data.country}</p>
    <p>위치: ${data.loc}</p>
    <p>조직: ${data.org}</p>
    <p>시간대: ${data.timezone}</p>
    `;
})
.catch(error => {
    console.error('데이터를 가져오는 중 오류 발생:', error);
    document.getElementById('sf-ipinfo-result').innerHTML = `<p style="color: red;">데이터를 가져오는 중 오류가 발생했습니다.</p>`;
});

    throw new Error('네트워크 응답이 좋지 않습니다.');
    }

요금: 무료 (제한) 및 유료 플랜 제공

특징: 정확한 데이터 제공

DB-IP 사용 예시 (JavaScript)

다음은 JavaScript를 사용하여 DB-IP API를 호출하고 IP 주소 정보를 가져오는 예제입니다.

                
const apiKey = 'YOUR_API_KEY'; // 여기에 API 키를 입력하세요
fetch(`https://api.db-ip.com/v2/free/self?key=${apiKey}`)
.then(response => {
    if (!response.ok) {
    throw new Error('네트워크 응답이 좋지 않습니다.');
    }
    return response.json();
})
.then(data => {
    console.log('IP 주소 정보:', data);
    document.getElementById('sf-dbip-result').innerHTML = `
    <h4>결과 표시 영역</h4>
    <p>IP: ${data.ipAddress}</p>
    <p>국가: ${data.countryName}</p>
    <p>지역: ${data.stateProv}</p>
    <p>도시: ${data.city}</p>
    <p>위도: ${data.latitude}</p>
    <p>경도: ${data.longitude}</p>
    `;
})
.catch(error => {
    console.error('데이터를 가져오는 중 오류 발생:', error);
    document.getElementById('sf-dbip-result').innerHTML = `<p style="color: red;">데이터를 가져오는 중 오류가 발생했습니다.</p>`;
});

참고 사항

  • 일부 API는 사용량 제한이 있을 수 있습니다.
  • API 키가 필요한 경우, 해당 서비스에서 발급받아야 합니다.
  • 요금 정책은 각 서비스 제공사의 웹사이트에서 확인하시기 바랍니다.
  • ip-api.com은 CORS를 지원하지 않으므로, 클라이언트 측에서 직접 호출하는 경우 CORS 관련 오류가 발생할 수 있습니다. 이 경우 서버 측에서 API를 호출하여 데이터를 가져오는 방법을 고려해야 합니다.
